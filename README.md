# Livestock Logistics APP 🐄🚚

This is a real-time livestock shipment tracking app with role-based dashboards for:
- ✅ Senders (create, manage, confirm shipments)
- 🚛 Transporters (accept + track)
- 🛠 Admins (monitor all users and shipments)

## Tech Stack

- **Frontend**: React + Tailwind + Socket.IO
- **Backend**: Node.js + Express + MongoDB
- **Real-time**: WebSockets (Socket.IO)
- **Maps**: Google Maps JS API
